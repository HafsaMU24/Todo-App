#  Smart Inköpslista (MVP)

En enkel och kraftfull applikation för att hantera inköp, byggd med fokus på **prestanda** och **tillgänglighet**. Detta är en prototyp för lösningar riktade mot marknader under utveckling.

##  Varför det här projektet?
I många tillväxtländer är internethastighet och datakostnader en stor utmaning. Denna applikation är byggd med **Vanilla JavaScript** för att säkerställa:
- **Snabb laddning:** Minimal kodstorlek.
- **Hög effektivitet:** Smidig prestanda även på äldre mobiler eller svagare uppkopplingar.

##  Teknikstack
- **HTML5:** För en semantisk struktur.
- **CSS:** Responsiv design som fungerar på alla skärmstorlekar.
- **JavaScript (ES6+):** För logik och lokal datahantering.
- **GitHub Actions (CI/CD):** Konfigurerad Pipeline för automatisk publicering för att säkerställa snabba uppdateringar och hög kvalitet.

## ✨ Funktioner
-  Lägg till och ta bort varor i realtid.
-  Lokal lagring (LocalStorage) så att data sparas även om webbläsaren stängs.
-  Rent och intuitivt användargränssnitt (UX).

##  Framtidsplaner (Roadmap)
-  **AI-integration:** Implementera funktioner för att förutsäga inköp baserat på användarens vanor.
-  **Offline-läge:** Fullt stöd för arbete utan internet (PWA).
-  **Azure DevOps:** Flytta hela arbetsflödet till Azure för att tillämpa högre säkerhetsstandarder och automatiserade tester.
